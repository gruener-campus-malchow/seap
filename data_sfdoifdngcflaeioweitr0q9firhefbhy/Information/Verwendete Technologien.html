HTML - HyperText Markup Language - Hypertext BeschreibungsSprache

html - damit beginnt alles!
head - damit legen wir die Eigenschaften für das Dokument fest.
body - Hier kommen die eigentlichen Inhalte rein.
h1 ... - H steht für Heading, die Überschrift. Überschriften ordnen sich einander unter.
div - Damit werden Textbereiche eingefasst.
hr - Eine Waagerechte Linie.
p - Damit werden ebenfalls Textbereiche beziehungsweise Absätze eingefasst.

Prinzipiell findet man alle Informationen zu HTML auf SELF HTML

CSS - Cascading Style Sheets - Aufeinander Aufbauende AussehensDokumente

Wir benutzen einfach CSS, denn das Thema ist sehr komplex.

Prinzipiell findet man alle Informationen zu CSS ebenfalls auf SELF HTML

FTP - File Transfer Protokoll - Dateien Übertragungs Abmachung

Ein Protokoll beziehungsweise (ganz deutsch sinnlich) Abmachung, beschreibt ganz genau wie Computer miteindander "reden", wenn sie Dateien von A nach B übertragen wollen.

Wir benötigen diese Dinge, um Dateien auf den Server in Internet zu übertragen.

Ein FTP-Programm oder auch FTP-Client kann man im POD herunterladen.

Informationen dazu findet man in der Wikipedia

PHP - Pre Hypertext Processor - Hypertext Vorverarbeiter

Eine sehr mächtige Programmiersprache, die auf Servern (im Internet) läuft.
Viele Webseiten werden damit programmiert. So auch der POD. Das besondere ist, dass man der Webseite eine eigene "Logik" gibt.
Näheres dazu werden wir in den Aufgaben besprechen
